    [section] 4 7.3 — Trigonometric Substitution

    For some stubborn integrals, one can use a substitution technique that is different from ordinary \(u\)-substitution. Examples include the (((xref without ref, first/last, or provisional attribute (check spelling)))) and trigonometric substitution. In this section we focus on trigonometric substitution, often called “trig sub”.
    🔗
    Trigonometric substitution works by replacing a variable with a trigonometric function of a new variable in order to exploit Pythagorean identities such as \(\cos^2\theta = 1 - \sin^2\theta\text{.}\) For example, if
    \begin{equation*} x = \sin\theta, \end{equation*}
    then
    \begin{equation*} \sqrt{1 - x^2} = \sqrt{1 - \sin^2\theta} = \sqrt{\cos^2\theta} = \cos\theta. \end{equation*}
    🔗
    Trigonometric substitutions must be one-to-one. As a result, the trigonometric functions used have restricted domains. Only certain substitutions—such as those appearing in the table below— are valid. We will not focus heavily on these restrictions.

    [subsection] 4.1 Examples

    • \(\displaystyle \int \sqrt{4 - x^2}\,dx\)
      🔗
    • \(\displaystyle \int \frac{1}{\sqrt{4x^2 + 1}}\,dx\)
      🔗
    • \(\displaystyle \int \frac{1}{\sqrt{x^2 - 9}}\,dx\)
      🔗
    • \(\displaystyle \int_{\sqrt{3}}^2 \frac{\sqrt{x^2 - 3}}{x}\,dx\)
      🔗
    • \(\displaystyle \int \frac{1}{(x^2 + 1)^{3/2}}\,dx\)
      🔗
    • \(\displaystyle \int \frac{x}{\sqrt{x^2 + 6x + 12}}\,dx\)
      🔗
    • \(\displaystyle \int \frac{x^2}{\sqrt{2x - x^2}}\,dx\)
      🔗
