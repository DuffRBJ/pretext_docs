    [section] 6 7.5 — Strategies for Integration

    In this chapter you have studied several integration techniques that can be used together with the basic integration rules. However, knowing how to use techniques is not enough. You must also know when to use them. Integration is fundamentally a problem of recognition.
    🔗
    Often, a small change in an integrand can require a completely different technique, or even result in a function whose antiderivative is not elementary.
    🔗
    \begin{equation*} \begin{array}{ll} \int x \ln x\,dx = \frac{x^2}{2}\ln x - \frac{x^2}{4} + C & \text{Integration by Parts} \\[6pt] \int \frac{\ln x}{x}\,dx = \frac{(\ln x)^2}{2} + C & \text{Substitution with Power Rule} \\[6pt] \int \frac{1}{x \ln x}\,dx = \ln|\ln x| + C & \text{Substitution with Log Rule} \\[6pt] \int \frac{x}{\ln x}\,dx & \text{Not an elementary function} \end{array} \end{equation*}
    A prerequisite for applying any strategy is fluency with the basic integration formulas. A table of these formulas appears again below for reference.
    🔗

    [subsection] 6.1 Strategy for Integration

    If you do not immediately see how to evaluate an integral, the following four-step strategy is often effective.
    🔗
    1. Simplify the integrand if possible. Algebraic manipulation or trigonometric identities may simplify the integrand and reveal an obvious technique.
      \begin{equation*} \int \frac{\tan \theta}{\sec^2\theta}\,d\theta = \int \sin\theta\cos\theta\,d\theta = \frac12 \int \sin(2\theta)\,d\theta \end{equation*}
      \begin{equation*} \int (\sin x + \cos x)^2\,dx = \int (1 + 2\sin x \cos x)\,dx \end{equation*}
      🔗
    2. Look for an obvious substitution. Try to identify a function \(u = g(x)\) whose derivative appears in the integrand, up to a constant multiple.
      🔗
    3. Classify the integrand by form.
      • Trigonometric functions. Products of powers of trigonometric functions are handled using the methods of Section 7.2.
        🔗
      • Rational functions. Use partial fractions as described in Section 7.4.
        🔗
      • Products of algebraic and transcendental functions. Try integration by parts using guidance from Section 7.1.
        🔗
      • Radicals.
        • Use trigonometric substitution for expressions involving \(\sqrt{\pm x^2 \pm a^2}\text{.}\)
          🔗
        • Use rationalizing substitutions for expressions such as \(\sqrt[n]{ax + b}\text{.}\)
          🔗
        🔗
      🔗
    4. Try again. If the first three steps fail, remember that the two fundamental methods are substitution and integration by parts. Manipulating the integrand or combining techniques may be necessary.
      \begin{equation*} \int \frac{1}{1-\cos x}\,dx = \int \left(\csc^2 x + \csc x \cot x\right)\,dx \end{equation*}
      🔗
